# Instruction #
Afin de bien démarrer le jeu :
- Démarrer le serveur, en lançant JHeadOrTailServer.java
- Lancer autant de fois le client (JHeadOrTailClient.java) que le nombre de joueurs qui doivent jouer
(Il est impossible de faire une partie de plus de deux joueurs)
- Ensuite vous pouvez jouer sur les consoles clients des joueurs se sont connectés

A vous de jouer !
